## Practice Rules (Professional Workflow)

1. Read the concept note first.
2. Run each command manually (do not just copy-paste all at once).
3. Keep a small lab journal:
   - commands used
   - errors faced
   - how you fixed them
4. Re-run difficult labs after 1 week for retention.

---

## Completion Goal

After finishing this lab path, you should be able to:

- Navigate Linux confidently
- Manage files and permissions safely
- Troubleshoot common shell issues
- Solve basic Linux CTF-style tasks independently
